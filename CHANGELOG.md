# Changelog

All notable changes to Chorus will be documented here.

## [0.1.0] - 2026-03-28

### Initial Release
- Multi-client column view with real-time UDP log streaming
- Time-aligned snap rows — shared network events align horizontally across clients
- Color-coded PS: tag categories (NETWORK, STATE, INIT, RECONNECT, GAME)
- Filter pills by log type (log, warning, error, network)
- Ctrl+F search with next/previous navigation
- Pause and resume live log feed with buffering
- Save and load sessions as JSON
- Import Chrome DevTools browser log exports
- Client rename events — columns relabel without creating duplicates
- New Session and Clear controls
- Configurable snap window (ms)
- Windows executable via electron-builder
